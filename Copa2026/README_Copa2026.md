# ⚽ Copa do Mundo 2026 — Análise de Dados

Análise estatística das seleções da Copa do Mundo 2026 usando Python e Pandas, com geração de gráficos profissionais. Projeto sem API — dados reais extraídos do dataset oficial da FIFA.

---

## 📊 O que o projeto faz

- Lê e processa dados reais de 48 seleções com 136 variáveis cada
- Gera ranking das seleções que mais fizeram gols
- Compara ataque x defesa das top 10 seleções
- Identifica melhores defesas e piores campanhas
- Exporta gráficos em PNG prontos para apresentação

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| `Python` | Linguagem principal |
| `Pandas` | Leitura, filtragem e análise dos dados |
| `Matplotlib` | Geração dos gráficos |
